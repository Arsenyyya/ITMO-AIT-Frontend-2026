#Лабораторная работа 1 (вариант 4)

Тема: сервис для управления личными финансами и бюджетом.

##Что реализовано

- `login.html` — вход.
- `register.html` — регистрация.
- `dashboard.html` — личный кабинет пользователя (счета, транзакции, бюджеты).
- `transactions.html` — поиск и фильтрация транзакций по категории, сумме и дате.
- `report.html` — страница отчёта (расходы, категории, прогнозы).
- `integrations.html` — интеграция с платежными аккаунтами.
- `assets/css/styles.css` — кастомная тема и адаптивность поверх Bootstrap.
- `assets/js/script.js` — интерактив (валидация форм, фильтрация, отчёт, модальные окна).

##Технологии

- HTML5 (семантическая верстка)
- CSS3 + CSS variables
- Bootstrap 5.3
- JavaScript (DOM, события, localStorage)

##Как открыть

1. Открыть `index.html` в браузере.
2. Перейти на нужную страницу из навигации.
