# Домашняя работа 4 (вариант 4)

Тема: SVG-спрайт.

## Что сделано

- Создан единый SVG-спрайт: `assets/icons/sprite.svg`.
- В спрайт добавлено 5 иконок по тематике финансового сервиса:
  - `icon-wallet`
  - `icon-chart`
  - `icon-card`
  - `icon-target`
  - `icon-sync`
- На странице `index.html` показано использование всех иконок через `<use>`.

## Структура

- `index.html` — демонстрация использования спрайта.
- `assets/icons/sprite.svg` — файл спрайта с символами (`<symbol>`).
- `assets/css/styles.css` — стили страницы.

## Запуск

1. Открыть `index.html` через Live Server (или напрямую в браузере).
2. Проверить, что все 5 иконок отрисовываются из одного файла спрайта.
