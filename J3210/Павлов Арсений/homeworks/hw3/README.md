# Домашняя работа 3 (вариант 4)

Тема: CSS-переменные и темизация сайта.

## Что сделано

- На базе предыдущей версии сайта реализованы две темы:
  - `neon` (базовая тёмная);
  - `paper` (светлая).
- Все основные цветовые решения вынесены в CSS-переменные.
- Добавлен JS-переключатель темы в навигации на всех страницах.
- Выбранная тема сохраняется в `localStorage` (`hw3-theme`) и восстанавливается при перезагрузке.
- Тема применяется через `data-theme` на корневом элементе (`html[data-theme="..."]`).

## Структура

- `index.html`, `login.html`, `register.html`, `dashboard.html`, `transactions.html`, `report.html`, `integrations.html`.
- `assets/css/styles.css` — переменные тем и стили для двух визуальных режимов.
- `assets/js/script.js` — логика приложения + переключение/сохранение темы.
- `db.json`, `routes.json`, `package.json` — mock API для интерактивных страниц.

## Запуск

1. Перейти в папку:
   ```bash
   cd "J3210/Павлов Арсений/homeworks/hw3"
   ```
2. Установить зависимости:
   ```bash
   npm install
   ```
3. Запустить API:
   ```bash
   npm run api
   ```
4. Открыть сайт через Live Server (`index.html`).

## Проверка

1. Нажать кнопку переключения темы в навбаре.
2. Убедиться, что визуальный стиль меняется на всех страницах.
3. Обновить страницу — выбранная тема должна сохраниться.
4. Перейти между страницами (`dashboard`, `transactions`, `report`, `integrations`) — тема не должна сбрасываться.
